# Indian_Ecosystem
This project focuses on analyzing funding trends for Indian start-ups from 2018 to 2021. As the team's data expert, the goal is to explore the funding landscape and provide insights to guide our venture into the Indian start-up ecosystem

# Project Objectives:
Analyze funding trends in the Indian start-up ecosystem.
Identify the location with the highest concentration of start-ups.
Explore the distribution of funding across different cities.
Determine the sector receiving the highest funding.
Investigate the most common investment types.
Examine the number of start-ups funded each year.
